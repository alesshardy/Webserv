Webserv - Serveur Web Professionnel

Un serveur web moderne, performant et conçu pour répondre à vos besoins.

À propos de Webserv

Webserv est un serveur web robuste, capable de gérer plusieurs connexions simultanément. Il prend en charge les méthodes HTTP essentielles et offre des fonctionnalités avancées pour les développeurs et les entreprises.

Fonctionnalités

GET

Récupérez des ressources facilement.

POST

Envoyez des données au serveur.

DELETE

Supprimez des ressources sur le serveur.

UPLOAD

Téléversez des fichiers vers le serveur.

Python CGI

Exécutez des scripts Python CGI dynamiques.

PHP CGI

Exécutez des scripts PHP CGI dynamiques.

Comment démarrer ?

Lancez le serveur avec la commande suivante :

./webserv [fichier_de_configuration]

Ensuite, accédez au serveur via votre navigateur à l'adresse http://localhost:8080.
