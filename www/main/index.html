WEBSERV

Un serveur HTTP performant et flexible

À propos de Webserv

Webserv est un projet dans le cadre de l'école 42, conçu pour être un serveur HTTP performant et flexible. Il est capable de gérer des requêtes GET, POST et DELETE, ainsi que le téléversement de fichiers.

Fonctionnalités

GET

Récupérez des ressources facilement.

POST

Envoyez des données au serveur.

DELETE

Supprimez des ressources sur le serveur.

UPLOAD

Téléversez des fichiers vers le serveur.

Python CGI

Exécutez des scripts Python CGI dynamiques.

PHP CGI

Exécutez des scripts PHP CGI dynamiques.

Shell CGI

Exécutez des commandes shell via CGI.

Listing des répertoires

Affichez le contenu des répertoires.

Gestion des erreurs

Gérez les erreurs HTTP avec des pages personnalisées.

Redirection

Redirigez vers mediapart.fr.

CGI GET/POST

Recevez un compliment aléatoire en GET et POST

Comment démarrer ?

Lancez le serveur avec la commande suivante :

./webserv [fichier_de_configuration]

Ensuite, accédez au serveur via votre navigateur à l'adresse http:[ip/localhost]:[port].
